# Aloe World – Cosmetic Landing Page

## About the Project

**Aloe World** is a simple cosmetic brand landing page built using **HTML and CSS**.
The website showcases aloe-based skincare products and includes sections for products, reviews, and a contact form.

This project was created as a **front-end practice project** to improve layout design, Flexbox usage, animations, and basic UI structure.

---

## Features

* Sticky navigation bar
* Smooth section navigation using anchor links
* Animated hero section
* Product showcase cards
* Hover effects on buttons and reviews
* Font Awesome icons for UI elements
* Customer review section
* Contact form
* Simple responsive structure

---

## Technologies Used

* **HTML5**
* **CSS3**
* **Flexbox Layout**
* **Font Awesome Icons**
